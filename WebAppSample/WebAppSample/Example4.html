JavaScript Events

Drag your finger on this box and let go!

An alert will pop up once you trigger an animation event in the OnTouch Example section now that you've clicked the magic button.

Javascript events are quite fun! There are an insane amount of events available out there for just about anything you can think of. Many of these events can be added to the HTML portion of the code, such as 'onload' and 'onclick'. Some require an event listener to be added using JavaScript though, such as with the touch events. You can use any of these events to triger some JavaScript code, so it's especially useful in building dynamic user interfaces on a website.

CSS3 Transitions

CSS3 Transitions are pretty simple. They also provide a lot of flair for the user interface, especially if you're doing a dynamic interface or have lots of objects or items on your page. Transitions make changes to the DOM much smoother, which provides a better user experience.

  • https://www.w3schools.com/css/css3_transitions.asp
  • https://github.com/tinker-camren/CIT-261-Again/blob/master/WebAppSample/WebAppSample/Content/Site.css

CSS3 Animations

CSS3 Animations are much like transitions. They're good for sprucing up your page, especially if you have a lot of items on your page. It also helps when you have dynamic web pages that change a lot without the page reloading. There are many things you can animate on a page, include pictures. This example is just a basic one showcasing what can be done.

  • https://www.w3schools.com/css/css3_animations.asp
  • https://github.com/tinker-camren/CIT-261-Again/blob/master/WebAppSample/WebAppSample/Content/Site.css

CSS3 Transforms

CSS3 Transforms are much like the other two topics above. You can use them to transform elements on a web page. Transforms allow you to change the shape and position of element(s) without affecting the rest of the DOM.

  • https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Transforms/Using_CSS_transforms
